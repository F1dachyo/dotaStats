Для запуска в pycharm необходимо установить бибилиотеки (pip install -r requirements.txt).
Есть возможность запуска через exe файл (first/main.exe).

По умолчанию запускается вкладка с информацией об владельце (обновляется при запуске).
Вторая вкладка это поиск игрока по id (изначально загружается информация по последнему id (НЕ обновляется при запуске))

В настройках есть возможность указать id владельца, а также тему (светлая/тёмная).
